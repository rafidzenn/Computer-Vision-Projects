# others/

Per the course manual, this folder holds the non-code deliverables:

- `final_presentation.pptx` — final presentation slides
- `final_report.pdf` — 8-page final report (IEEE double-column)
- `update_presentation.pptx` — update presentation slides
- `update_report.pdf` — 2-page update report (IEEE double-column)
- `demo.mp4` — one-minute screen-capture video of the project demo run

These are added as each milestone approaches — they are intentionally not part
of the proof-of-concept code drop.
